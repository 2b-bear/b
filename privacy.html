b-bear Privacy

DISCLAIMER: With HTML files proxied through Cloudflare, we cannot disable Cloudflare Insights.

B-bear.org collects only one type of data: Cloudflare Insights. Cloudflare Insights is a analytical tool to see end-user's performance and other data.

This includes, and may not be limited to: visit counts, Internet Protocol (IP) addresses for security and geographic information about the user base, device and browser data, your network performance (to improve load times), referral data (if you came from another website), and bot detection.

These are the services we use, so please read their Privacy Policies as well: Cloudflare, Vercel, and GitHub.
